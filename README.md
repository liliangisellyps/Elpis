## Installation

```bash
# Clone este repositório
$ git clone https://github.com/liliangisellyps/Elpis

# Instale as dependências
$ npm install

# Execute a aplicação
$ npm run serve

# O servidor iniciará na porta:8080 - acesse http://localhost:8080 
```
